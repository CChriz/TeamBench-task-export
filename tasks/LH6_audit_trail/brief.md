# LH6: Audit Trail Bug Fixes (Brief)

Fix 5 bugs in a Python web application and log every fix in `AUDIT_LOG.md`.
Bugs: SQL injection, XSS, missing auth check, pagination off-by-one, broken CSV export.

Each fix must have a structured audit entry in AUDIT_LOG.md:
```
## Fix N: <title>
- **File**: <path>
- **Bug**: <description>
- **Fix**: <what changed>
- **Risk**: low|medium|high
```

Pass = 4+ bugs fixed AND 4+ valid audit entries.
