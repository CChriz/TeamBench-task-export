# TEST5: Mutation-Resistant Test Suite (Brief)

Write comprehensive pytest tests for the math library in `tests/test_mathlib.py`.
The library `mathlib.py` has 25 functions/behaviors. Your tests must:
- Pass on the correct implementation
- Kill at least 16 out of 20 mutant versions (each has one subtle bug)
- Include at least 25 test functions covering all behaviors and edge cases
Run: `python -m pytest tests/test_mathlib.py -v`
